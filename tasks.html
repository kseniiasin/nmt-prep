Квадратні рівняння

Середній рівень

Розв'яжіть рівняння: 2x² - 5x - 3 = 0

Знаходимо дискримінант: D = b² - 4ac
D = (-5)² - 4·2·(-3) = 25 + 24 = 49
Оскільки D > 0, рівняння має два корені:
x₁ = (5 + √49)/4 = (5 + 7)/4 = 3
x₂ = (5 - √49)/4 = (5 - 7)/4 = -0.5
Відповідь: x₁ = 3, x₂ = -0.5

Теорема Піфагора

Легкий рівень

У прямокутному трикутнику один катет дорівнює 6 см, а гіпотенуза - 10 см. Знайдіть довжину другого катета.

За теоремою Піфагора: c² = a² + b²
Виражаємо невідомий катет: b = √(c² - a²)
b = √(10² - 6²) = √(100 - 36) = √64 = 8 см
Відповідь: 8 см

Складний рівень

Спростити вираз: (3x²y⁻³)³ / (9x⁻¹y²)²

Піднесемо чисельник і знаменник до степеня:
(3x²y⁻³)³ = 27x⁶y⁻⁹
(9x⁻¹y²)² = 81x⁻²y⁴
Ділення: (27/81)x6-(-2)y-9-4
= (1/3)x⁸y⁻¹³ = x⁸ / (3y¹³)
Відповідь: x⁸ / (3y¹³)
